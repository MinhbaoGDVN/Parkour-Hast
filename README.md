**PARKOUR HAST - OFFICIAL WEBSITE**

The official website for **Parkour Hast**, an international Minecraft parkour server. Built with a clean, modern, and high-performance layout featuring a Green & White aesthetic.

[![Deployment Status](https://img.shields.io/badge/Deploy-Vercel-000000?style=for-the-badge&logo=vercel&logoColor=white)](https://vercel.com)
[![Minecraft Version](https://img.shields.io/badge/Minecraft-26.1.2-22c55e?style=for-the-badge&logo=minecraft&logoColor=white)](https://minecraft.net)

---

## 🌟 Features

- 🎨 **Modern Vibe:** Minimalist interface with clean animations, heavily inspired by modern studio designs.
- 🌎 **Global Friendly:** Fully written in English to welcome players worldwide.
- 🖱️ **Smooth Navigation:** Fluid scrolling effects between pages with a blurred navbar effect (Glassmorphism).
- 📋 **One-click Copy IP:** Players can instantly copy the server IP with dynamic button feedback.

---

## 🛠️ Built With

- **HTML5 & CSS3** - Responsive design, custom animations, and clean grid system.
- **Vanilla JavaScript** - Clipboard API handling and smooth navigation.
- **Vercel** - Fast and reliable global hosting.
